はじめに

Chrome Book やタブレットではディスク容量が小さく、動画や大量の音楽ファイルを持つことができません。 家庭内で一人で使うことを前提にこれらの低性能マシンでもメインマシンの大量の動画ファイルなどを利用できるようにしたいと思って開発してみました。

そのため、ログイン画面や排他処理のようなことはしていません。

 

インストール

動作条件

下記のものがインストールされていること。

  • Node.js 10.0 以上
  • MySQL 5.x または MariaDB 10.x

インストール手順

GitHub から clone を行います。

git clone https://github.com/makandat/express

express フォルダに MyPC フォルダがあるのでそれをそのまま使ってもいいし、どこかへ移動しても構いません。他のフォルダは不要なので削除して構いません。

MyPC フォルダへ移動して必要なモジュールをインストールします。

npm install
