AWS S3 ヘルプ

AWS S3 を使うための準備

AWS とは Amazon Web Service の略で S3 は Simple Storage Service の略です。

S3 を使うためには AWS マネージメントコンソールから IAM にアクセスして認証情報を作成するか、前にダウンロードしておいた認証情報が必要になります。

具体的にはアクセスキーと秘密キーが必要になります。

AWS CLI をインストールしてある場合、これらはファイル ~/.aws/credentials (Linux) あるいは C:/Users/user/.aws/credentials (Windows) に含まれています。

そうでない場合は、自身で保存しておいた認証情報ファイルから取得します。

MyPC フォルダ内に aws_config.json というファイルをエディタで作成します。その内容は次の例のようにします。

                { "accessKeyId": "AKIAII*******ASJBW45A", "secretAccessKey": "hb3yJrVOD*******rNPK/7ir6Mae*****miaqD8w", "region": "******" }

accessKeyId と secretAccessKey は credentials からコピーします。region はバケットを作った場所で、例えば "us-east-1" とかです。
